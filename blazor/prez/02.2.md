## Запуск

Через **`dotnet-cli`**:

```cmd
cd BlazorApp1
dotnet run
```

После запуска у вас появится возможность перейти в браузере по адресу https://localhost:5001 и увидеть свое первое **`Blazor WebAssembly`**-приложение:
