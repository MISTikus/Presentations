## Изоляция стилей

---

##### `MyComponent.razor`

```html
<span class="my-class">Текст с красным фоном</span>
```

```css
.my-class {
  background-color: red;
}
```

```html
<span class="my-class">Текст без фона</span>
```

##### `Index.razor`

```csharp

<MyComponent />
<OtherComponent />
```
